# smart-planner
Smart Daily Planner with Priority & Progress Tracker

 Overview
The Smart Daily Planner is a simple yet effective productivity tool built using Python in Jupyter Notebook. It helps users organize daily tasks, assign priorities, and track their progress in a structured way.

This project focuses on improving time management and productivity through a clean and easy-to-use system.

Problem Statement
Many people struggle to manage daily tasks efficiently. Tasks are often forgotten or completed without proper prioritization, leading to reduced productivity.

Solution
This project provides a task management system where users can:
- Add tasks
- Assign priority levels
- View tasks sorted by importance
- Mark tasks as completed
- Track overall progress

Features
-  Add tasks interactively
-  Assign priority (High / Medium / Low)
-  Automatic sorting based on priority
-  Mark tasks as completed
-  Progress tracking (percentage)
-  Motivational feedback based on performance

Technologies Used
- Python
- Jupyter Notebook

How to Run the Project
1. Open JupyterLab
2. Open the file: "Smart_Planner.ipynb"
3. Run all cells step by step (Shift + Enter)
4. Follow the instructions in the notebook

Example Output
- Task list displayed with priority
- Completed tasks marked with ✔
- Progress report showing completion percentage
- Feedback message based on productivity

Project Logic (Simple Explanation)
The system uses:
- A list to store tasks
- Priority-based sorting
- Boolean values to track completion
- Basic calculations to measure progress
